  ## Einstieg
</div>

<div id="using-reference-servers">
  ### Verwendung von Referenzservern
</div>

TypeScript-basierte Server können direkt mit `npx` verwendet werden:

```bash
npx -y @modelcontextprotocol/server-memory
```

Python-basierte Server können mit `uvx` (empfohlen) oder `pip` verwendet werden:

```bash
# Mit uvx
uvx mcp-server-git

# Mit pip
pip install mcp-server-git
python -m mcp_server_git
```

<div id="configuring-with-claude">
  ### Konfiguration mit Claude
</div>

Um einen MCP-Server mit Claude zu verwenden, fügen Sie ihn Ihrer Konfiguration hinzu:

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/path/to/allowed/files"
      ]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
