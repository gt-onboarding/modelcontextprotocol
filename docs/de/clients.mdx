<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) ist ein Open-Source-Desktopassistent mit KI, plattformübergreifend, der Werkzeuge über MCP-Server unterstützt.

**Hauptfunktionen:**

- Integrierte MCP-Server lassen sich schnell aktivieren und deaktivieren.
- Nutzer können weitere Server hinzufügen, indem sie die Konfigurationsdatei anpassen.
- Open Source und benutzerfreundlich, geeignet für Einsteiger.
- Die MCP-Unterstützung wird fortlaufend ausgebaut.

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) ist eine Rust-Bibliothek, die die Erstellung von KI-Agenten vereinfacht. Die Bibliothek bietet eine nahtlose Integration mit MCP-Servern.

[Beispiel für die Integration eines MCP-Servers](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**Wichtige Funktionen:**

- Multi-LLM – Wir unterstützen die meisten LLM-APIs (OpenAI, Anthropic, Gemini, Ollama und alle mit der OpenAI-API kompatiblen).
- Integrierte Unterstützung für MCP-Server.
- Erstellen Sie Agenten-Workflows in einer typen- und speichersicheren Sprache wie Rust.

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) ist eine No-Code-AI-Plattform, mit der Sie rund um die Uhr Agents für Vertriebs-, Marketing- und Kreativaufgaben erstellen können. Verbinden Sie über MCP sicher mehr als 2.500 APIs und 10.000 Werkzeuge.

**Wichtigste Funktionen:**

- No-Code-Erstellung von AI-Agents und Aufbau von Workflows.
- Zugriff auf eine umfangreiche Bibliothek mit über 10.000 Werkzeugen und mehr als 2.500 APIs über MCP.
- Einfacher Dreischritt-Prozess zum Verbinden von MCP-Servern.
- Verbindungen sicher verwalten und Zugriff jederzeit widerrufen.

**Erfahren Sie mehr:**

- [AgenticFlow MCP-Integration](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] ist eine native, plattformübergreifende Desktop-Chat-Anwendung für KI mit MCP-Unterstützung. Sie unterstützt mehrere KI-Anbieter (z. B. Anthropic, Cloudflare, Deepseek, OpenAI, Qwen), lokale KI-Modelle (über vLLM, Ray usw.) sowie aggregierte API-Plattformen (wie Deepinfra, Openrouter u. a.).

**Wesentliche Funktionen:**

- **Dynamisches Umschalten von LLM-APIs und Agenten**: Nahtlos zwischen unterschiedlichen LLM-APIs und Agenten wechseln – jederzeit.
- **Umfassende Unterstützung von Fähigkeiten**: Integrierte Unterstützung für Werkzeuge, Prompts, Ressourcen und Sampling-Methoden.
- **Konfigurierbare Agenten**: Hohe Flexibilität dank auswählbarer und anpassbarer Werkzeuge über die Agenten-Einstellungen.
- **Erweiterte Sampling-Kontrolle**: Sampling-Parameter anpassen und mehrstufiges Sampling für optimale Ergebnisse nutzen.
- **Plattformübergreifende Kompatibilität**: Voll kompatibel mit macOS, Windows und Linux.
- **Kostenlos & Open Source (FOSS)**: Freizügige Lizenz erlaubt Modifikationen und individuelles App-Bundling.

**Mehr erfahren:**

- [TUUI-Dokumentation](https://www.tuui.com/)
- [AIQL-GitHub-Repository](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) ist ein Open-Source, agentischer Coding-Assistent für Terminals.

**Wichtige Funktionen:**

- Volle Unterstützung für MCP-Server.
- Prompts mit Ihrem bevorzugten Texteditor bearbeiten.
- Gespeicherte Prompts sofort mit `@` aufrufen.
- AWS-Ressourcen direkt über Ihr Terminal steuern und organisieren.
- Werkzeuge, Profile, Kontextverwaltung, Auto-Compacting und vieles mehr!

**Erste Schritte**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) ist ein Open-Source, agentischer Coding-Assistent für IDEs.

**Wichtige Funktionen:**

- Unterstützung für die IDEs VS Code, JetBrains, Visual Studio und Eclipse.
- AWS-Ressourcen direkt aus Ihrer IDE steuern und organisieren.
- Berechtigungen für jedes MCP-Werkzeug über die Benutzeroberfläche der IDE verwalten.

<div id="apify-mcp-tester">
  ### Apify MCP Tester
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) ist ein Open-Source-Client, der über Server-Sent Events (SSE) eine Verbindung zu jedem MCP-Server herstellt.
Er ist ein eigenständiger Apify-Actor, der für das Testen von MCP-Servern über SSE entwickelt wurde und Authorization-Header unterstützt.
Er verwendet reines JavaScript (Old-School-Stil) und wird auf Apify gehostet, sodass Sie ihn ohne Einrichtung ausführen können.

**Hauptfunktionen:**

- Stellt über SSE eine Verbindung zu jedem MCP-Server her.
- Funktioniert mit dem [Apify MCP Server](https://apify.com/apify/actors-mcp-server), um mit einem oder mehreren Apify-[Actors](https://apify.com/store) zu interagieren.
- Nutzt dynamisch Werkzeuge basierend auf Kontext und Benutzeranfragen (falls vom Server unterstützt).

<div id="amp">
  ### Amp
</div>

[Amp](https://ampcode.com) ist ein agentenbasiertes Coding-Werkzeug von Sourcegraph. Es läuft in VS Code (und kompatiblen Forks wie Cursor, Windsurf und VSCodium) sowie als Kommandozeilen-Tool. Es ist außerdem für Mehrbenutzerbetrieb ausgelegt — Sie können Threads teilen und mit Ihrem Team zusammenarbeiten.

**Wesentliche Funktionen:**

- Feingranulare Kontrolle über aktivierte Werkzeuge und Berechtigungen
- Unterstützung für MCP-Server, die in der VS-Code-Datei `mcp.json` definiert sind

<div id="augment-code">
  ### Augment Code

- **Konfigurierbarer Start**: MCP-Server so einstellen, dass sie automatisch mit Warp starten oder bei Bedarf manuell starten
