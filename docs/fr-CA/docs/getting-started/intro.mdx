---
title: Qu’est-ce que le Model Context Protocol (MCP)?
sidebarTitle: Qu’est-ce que MCP?
---

Le MCP (Model Context Protocol) est une norme open source qui permet de connecter des applications d’IA à des systèmes externes.

Avec le MCP, des applications d’IA comme Claude ou ChatGPT peuvent se connecter à des sources de données (p. ex. fichiers locaux, bases de données), à des outils (p. ex. moteurs de recherche, calculatrices) et à des flux de travail (p. ex. invites spécialisées), leur donnant accès à des informations clés et leur permettant d’exécuter des tâches.

Pensez au MCP comme à un port USB-C pour les applications d’IA. Tout comme l’USB-C offre une façon normalisée de connecter des appareils électroniques, le MCP offre une façon normalisée de connecter des applications d’IA à des systèmes externes.

<div id="what-can-mcp-enable">
  ## Que peut permettre MCP?
</div>

* Les agents peuvent accéder à votre Google Calendar et à Notion, et agir comme un assistant IA plus personnalisé.
* Claude Code peut générer une application Web complète à partir d’une maquette Figma.
* Les robots conversationnels d’entreprise peuvent se connecter à plusieurs bases de données au sein d’une organisation, permettant aux utilisateurs d’analyser des données par clavardage.
* Les modèles d’IA peuvent créer des modèles 3D dans Blender et les imprimer à l’aide d’une imprimante 3D.

<div id="why-does-mcp-matter">
  ## Pourquoi le MCP est-il important?
</div>

Selon votre rôle dans l’écosystème, le MCP peut offrir divers avantages.

* **Développeurs** : le MCP réduit le temps de développement et la complexité lors de la création d’une application ou d’un agent d’IA, ou de leur intégration.
* **Applications ou agents d’IA** : le MCP donne accès à un écosystème de sources de données, d’outils et d’applications qui améliorent les capacités et l’expérience des utilisateurs finaux.
* **Utilisateurs finaux** : le MCP permet des applications et des agents d’IA plus performants, capables d’accéder à vos données et d’agir en votre nom au besoin.
