  ## Détails du client
</div>

<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) est un assistant IA de bureau multiplateforme open source qui prend en charge les Outils via des Serveurs MCP.

**Fonctionnalités clés :**

- Des Serveurs MCP intégrés peuvent être activés et désactivés rapidement.
- Les utilisateurs peuvent ajouter d'autres serveurs en modifiant le fichier de configuration.
- Il est open source et convivial, idéal pour les débutants.
- La prise en charge future de MCP sera continuellement améliorée.

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) est une bibliothèque Rust conçue pour simplifier la création d’agents d’IA. La bibliothèque offre une intégration transparente avec les Serveurs MCP.

[Exemple d’intégration d’un Serveur MCP](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**Principales fonctionnalités :**

- Multi-LLM – Nous prenons en charge la plupart des API LLM (OpenAI, Anthropic, Gemini, Ollama et toutes celles compatibles avec l’API d’OpenAI).
- Prise en charge intégrée des Serveurs MCP.
- Créez des flux agentiques dans un langage sûr pour les types et la mémoire comme Rust.

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) est une plateforme d’IA sans code qui vous aide à créer des agents capables de gérer des tâches de vente, de marketing et de création 24 h sur 24. Connectez plus de 2 500 API et plus de 10 000 outils en toute sécurité via le MCP.

**Fonctionnalités clés :**

- Création d’agents d’IA sans code et conception de flux de travail.
- Accès à une vaste bibliothèque de plus de 10 000 outils et de plus de 2 500 API via le MCP.
- Processus simple en trois étapes pour connecter des Serveurs MCP.
- Gérez les connexions en toute sécurité et révoquez l’accès en tout temps.

**En savoir plus :**

- [Intégration MCP d’AgenticFlow](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] est une application de clavardage IA native et multiplateforme pour ordinateur de bureau avec prise en charge de MCP. Elle prend en charge plusieurs fournisseurs d’IA (p. ex., Anthropic, Cloudflare, Deepseek, OpenAI, Qwen), des modèles d’IA locaux (via vLLM, Ray, etc.) et des plateformes d’API agrégées (comme Deepinfra, Openrouter, et plus).

**Fonctionnalités clés :**

- **Basculement dynamique des API LLM et des agents** : Passez d’une API LLM ou d’un agent à l’autre en toute fluidité, à la volée.
- **Prise en charge complète des capacités** : Prise en charge intégrée des Outils, Invite, Ressources et méthodes d’Échantillonnage.
- **Agents configurables** : Flexibilité accrue avec des outils sélectionnables et personnalisables via les paramètres des agents.
- **Contrôle avancé de l’échantillonnage** : Modifiez les paramètres d’échantillonnage et exploitez l’échantillonnage multi-ronde pour des résultats optimaux.
- **Compatibilité multiplateforme** : Entièrement compatible avec macOS, Windows et Linux.
- **Gratuit et open source (FOSS)** : Une licence permissive permet les modifications et la création de forfaits d’apps personnalisées.

**En savoir plus :**

- [Documentation TUUI](https://www.tuui.com/)
- [Dépôt GitHub d’AIQL](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) est un assistant de codage agentif open source pour terminaux.

**Fonctionnalités clés :**

- Prise en charge complète des Serveurs MCP.
- Modifiez les invites avec votre éditeur de texte préféré.
- Accédez instantanément aux invites enregistrées avec `@`.
- Contrôlez et organisez les Ressources AWS directement depuis votre terminal.
- Outils, profils, gestion du contexte, compression automatique, et bien plus encore !

**Pour commencer**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) est un assistant de programmation agentique open source pour les IDE.

**Fonctionnalités clés :**

- Prise en charge des IDE VSCode, JetBrains, Visual Studio et Eclipse.
- Contrôlez et organisez les Ressources AWS directement à partir de votre IDE.
- Gérez les autorisations pour chaque Outils MCP via l’interface utilisateur de l’IDE.

<div id="apify-mcp-tester">
  ### Apify MCP Tester
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) est un client open source qui se connecte à n’importe quel Serveur MCP au moyen des Événements envoyés par le serveur (SSE).
C’est un Apify Actor autonome conçu pour tester des Serveurs MCP via SSE, avec prise en charge des en-têtes d’autorisation.
Il utilise du JavaScript simple (à l’ancienne) et est hébergé sur Apify, ce qui vous permet de l’exécuter sans configuration.

**Principales fonctionnalités :**

- Se connecte à n’importe quel Serveur MCP via SSE.
- Fonctionne avec l’[Apify MCP Server](https://apify.com/apify/actors-mcp-server) pour interagir avec un ou plusieurs [Actors](https://apify.com/store) Apify.
- Utilise dynamiquement des Outils en fonction du contexte et des requêtes de l’utilisateur (si pris en charge par le serveur).

<div id="amp">
  ### Amp
</div>

[Amp](https://ampcode.com) est un outil de programmation agentique conçu par Sourcegraph. Il fonctionne dans VS Code (et des dérivés compatibles comme Cursor, Windsurf et VSCodium) ainsi qu’en ligne de commande. Il est aussi collaboratif — vous pouvez partager des fils de discussion et travailler avec votre équipe.

**Fonctionnalités clés :**

- Contrôle granulaire des Outils activés et des autorisations
- Prise en charge des Serveurs MCP définis dans le `mcp.json` de VS Code

<div id="augment-code">
  ### Augment Code

[Zed](https://zed.dev/docs/assistant/model-context-protocol) est un éditeur de code haute performance avec prise en charge intégrée de MCP, axé sur les modèles d’invite et l’intégration d’outils.

**Principales fonctionnalités :**

- Les modèles d’invite sont proposés comme commandes précédées d’une barre oblique dans l’éditeur
- Intégration d’outils pour améliorer les flux de travail de développement
- Intégration étroite aux fonctionnalités de l’éditeur et au contexte de l’espace de travail
- Ne prend pas en charge les Ressources MCP

<div id="zencoder">
  ### Zencoder
</div>

[Zencoder](https://zecoder.ai) est un agent de codage offert comme extension pour VS Code et la famille d’EDI JetBrains, qui rejoint les développeurs là où ils travaillent déjà. Il comprend RepoGrokking (compréhension contextuelle approfondie des bases de code), un pipeline agentique, ainsi que la possibilité de créer et de partager des agents personnalisés.

**Fonctionnalités clés :**

- RepoGrokking — compréhension contextuelle approfondie des bases de code
- Pipeline agentique — exécute, teste et valide le code avant de produire le résultat
- Plateforme Zen Agents — possibilité de créer des agents personnalisés et de les partager avec l’équipe
- Bibliothèque d’Outils MCP intégrée, avec installation en un clic
- Agents spécialisés pour les tests unitaires et de bout en bout (E2E)

**En savoir plus :**

- [Documentation Zencoder](https://docs.zencoder.ai)

<div id="adding-mcp-support-to-your-application">
  ## Ajout de la prise en charge de MCP à votre application
</div>

Si vous avez ajouté la prise en charge de MCP à votre application, nous vous encourageons à soumettre une demande d’extraction pour l’ajouter à cette liste. L’intégration de MCP peut offrir à vos utilisateurs de puissantes capacités d’IA contextuelles et faire de votre application un élément de l’écosystème MCP en pleine croissance.

Avantages d’ajouter la prise en charge de MCP :

- Permettre aux utilisateurs d’apporter leur propre contexte et leurs outils
- Rejoindre un écosystème grandissant d’applications d’IA interopérables
- Offrir aux utilisateurs des options d’intégration flexibles
- Prendre en charge des flux de travail d’IA « local-first »

Pour commencer à implémenter MCP dans votre application, consultez notre documentation [Python](https://github.com/modelcontextprotocol/python-sdk) ou [SDK TypeScript](https://github.com/modelcontextprotocol/typescript-sdk)
