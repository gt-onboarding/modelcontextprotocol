<Info>**Revisión del protocolo**: 2024-11-05</Info>

Los servidores proporcionan los bloques de construcción fundamentales para agregar contexto a los modelos de lenguaje mediante
MCP. Estas primitivas habilitan interacciones enriquecidas entre clientes, servidores y modelos de lenguaje:

* **Indicaciones**: Plantillas o instrucciones predefinidas que orientan las interacciones del modelo de lenguaje
* **Recursos**: Datos o contenido estructurado que aportan contexto adicional al modelo
* **Herramientas**: Funciones ejecutables que permiten a los modelos realizar acciones o obtener información

Cada primitiva puede resumirse en la siguiente jerarquía de control:

| Primitiva | Control                | Descripción                                        | Ejemplo                         |
| --------- | ---------------------- | -------------------------------------------------- | ------------------------------- |
| Indicaciones   | Control del usuario        | Plantillas interactivas invocadas por decisión del usuario       | Comandos con barra, opciones de menú    |
| Recursos | Control de la aplicación | Datos contextuales adjuntos y gestionados por el cliente | Contenido de archivos, historial de Git      |
| Herramientas     | Control del modelo       | Funciones expuestas al LLM para ejecutar acciones       | Solicitudes POST a una API, escritura de archivos |
