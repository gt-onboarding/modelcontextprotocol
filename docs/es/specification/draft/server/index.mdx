<Info>**Revisión del protocolo**: borrador</Info>

Los servidores proporcionan los componentes fundamentales para añadir contexto a los modelos de lenguaje mediante
MCP. Estos primitivos permiten interacciones más ricas entre clientes, servidores y modelos de lenguaje:

* **Indicaciones**: Plantillas o instrucciones predefinidas que guían las interacciones del modelo de lenguaje
* **Recursos**: Datos o contenido estructurado que aporta contexto adicional al modelo
* **Herramientas**: Funciones ejecutables que permiten a los modelos realizar acciones o obtener información

Cada primitivo puede resumirse en la siguiente jerarquía de control:

| Primitivo | Control                | Descripción                                        | Ejemplo                         |
| --------- | ---------------------- | -------------------------------------------------- | ------------------------------- |
| Indicaciones   | Controlado por el usuario        | Plantillas interactivas invocadas por elección del usuario       | Comandos con barra diagonal, opciones de menú    |
| Recursos | Controlado por la aplicación | Datos contextuales adjuntos y gestionados por el cliente | Contenido de archivos, historial de git      |
| Herramientas     | Controlado por el modelo       | Funciones expuestas al LLM para realizar acciones       | Solicitudes POST a API, escritura de archivos |
