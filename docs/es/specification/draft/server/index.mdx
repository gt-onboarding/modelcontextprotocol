<Info>**Revisión del protocolo**: borrador</Info>

Los servidores proporcionan los componentes fundamentales para agregar contexto a los modelos de lenguaje mediante
MCP. Estos primitivos habilitan interacciones más ricas entre clientes, servidores y modelos de
lenguaje:

* **Indicaciones**: Plantillas o instrucciones predefinidas que guían las interacciones del modelo de lenguaje
* **Recursos**: Datos estructurados o contenido que proporcionan contexto adicional al modelo
* **Herramientas**: Funciones ejecutables que permiten a los modelos realizar acciones o recuperar
  información

Cada primitivo puede resumirse en la siguiente jerarquía de control:

| Primitivo | Control                | Descripción                                        | Ejemplo                         |
| --------- | ---------------------- | -------------------------------------------------- | ------------------------------- |
| Indicaciones   | Control del usuario        | Plantillas interactivas invocadas por elección del usuario       | Comandos con barra, opciones de menú    |
| Recursos | Control de la aplicación | Datos contextuales adjuntos y gestionados por el cliente | Contenido de archivos, historial de Git      |
| Herramientas     | Control del modelo       | Funciones expuestas al LLM para realizar acciones       | Solicitudes POST a API, escritura de archivos |
