title: Descripción general
---

<Info>**Revisión del protocolo**: 2025-03-26</Info>

Los servidores proporcionan los bloques de construcción fundamentales para añadir contexto a los modelos de lenguaje mediante
MCP. Estos elementos primitivos habilitan interacciones ricas entre clientes, servidores y modelos de
lenguaje:

* **Indicaciones**: Plantillas o instrucciones predefinidas que guían las interacciones del modelo de lenguaje
* **Recursos**: Datos o contenido estructurado que aportan contexto adicional al modelo
* **Herramientas**: Funciones ejecutables que permiten a los modelos realizar acciones o
  obtener información

Cada elemento primitivo puede resumirse en la siguiente jerarquía de control:

| Primitivo | Control                | Descripción                                        | Ejemplo                         |
| --------- | ---------------------- | -------------------------------------------------- | ------------------------------- |
| Indicaciones | Controlado por el usuario        | Plantillas interactivas invocadas por elección del usuario       | Comandos con barra diagonal, opciones de menú |
| Recursos    | Controlado por la aplicación | Datos contextuales adjuntos y gestionados por el cliente | Contenido de archivos, historial de Git      |
| Herramientas | Controlado por el modelo       | Funciones expuestas al LLM para realizar acciones       | Solicitudes POST a APIs, escritura de archivos |
