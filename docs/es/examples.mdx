Esta página presenta varios servidores del Protocolo de Contexto de Modelo (MCP) que demuestran las capacidades y la versatilidad del protocolo. Estos servidores permiten que los modelos de lenguaje de gran tamaño (LLMs) accedan de forma segura a Herramientas y fuentes de datos.

<div id="reference-implementations">
  ## Implementaciones de referencia
</div>

Estos servidores de referencia oficiales muestran las funciones principales del MCP y el uso del SDK:

<div id="current-reference-servers">
  ### Servidores de referencia actuales
</div>

- **[Everything](https://github.com/modelcontextprotocol/servers/tree/main/src/everything)** - Servidor de referencia y pruebas con Indicaciones, Recursos y Herramientas
- **[Fetch](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)** - Obtención y conversión de contenido web para un uso eficiente de modelos LLM
- **[Filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)** - Operaciones de archivos seguras con controles de acceso configurables
- **[Git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)** - Herramientas para leer, buscar y manipular repositorios Git
- **[Memory](https://github.com/modelcontextprotocol/servers/tree/main/src/memory)** - Sistema de memoria persistente basado en grafos de conocimiento
- **[Sequential Thinking](https://github.com/modelcontextprotocol/servers/tree/main/src/sequentialthinking)** - Resolución de problemas dinámica y reflexiva mediante secuencias de pensamiento
- **[Time](https://github.com/modelcontextprotocol/servers/tree/main/src/time)** - Capacidades de conversión de hora y zona horaria

  ## Primeros pasos
</div>

<div id="using-reference-servers">
  ### Uso de servidores de referencia
</div>

Los servidores basados en TypeScript se pueden utilizar directamente con `npx`:

```bash
npx -y @modelcontextprotocol/server-memory
```

Los servidores basados en Python se pueden utilizar con `uvx` (recomendado) o `pip`:

```bash
# Con uvx
uvx mcp-server-git

# Con pip
pip install mcp-server-git
python -m mcp_server_git
```

<div id="configuring-with-claude">
  ### Configuración con Claude
</div>

Para usar un Servidor MCP con Claude, añádelo a tu configuración:

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/path/to/allowed/files"
      ]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
