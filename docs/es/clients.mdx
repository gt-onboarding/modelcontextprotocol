  ## Detalles del Cliente

<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) es un asistente de IA de escritorio multiplataforma y de código abierto que admite Herramientas a través de Servidores MCP.

**Características clave:**

- Los Servidores MCP integrados se pueden activar y desactivar rápidamente.
- Los usuarios pueden añadir más servidores modificando el archivo de configuración.
- Es de código abierto y fácil de usar, ideal para principiantes.
- Se mejorará continuamente la compatibilidad con MCP.

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) es una biblioteca de Rust diseñada para simplificar la creación de agentes de IA. La biblioteca incluye integración fluida con Servidores MCP.

[Ejemplo de integración con Servidor MCP](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**Características clave:**

- Multi-LLM: compatible con la mayoría de las API de LLM (OpenAI, Anthropic, Gemini, Ollama y todas las compatibles con la API de OpenAI).
- Compatibilidad integrada con Servidores MCP.
- Crea flujos de agentes en un lenguaje con seguridad de tipos y memoria como Rust.

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) es una plataforma de IA sin código que te ayuda a crear agentes que gestionan ventas, marketing y tareas creativas las 24 horas. Conecta más de 2,500 API y más de 10,000 herramientas de forma segura vía MCP.

**Funciones clave:**

- Creación de agentes de IA sin código y diseño de flujos de trabajo.
- Accede a una amplia biblioteca de más de 10,000 herramientas y más de 2,500 API a través de MCP.
- Proceso sencillo de 3 pasos para conectar Servidores MCP.
- Gestiona conexiones de forma segura y revoca el acceso en cualquier momento.

**Más información:**

- [Integración de AgenticFlow con MCP](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] es una aplicación de chat de IA de escritorio, nativa y multiplataforma, con soporte para el Protocolo de Contexto del Modelo (MCP). Admite múltiples proveedores de IA (p. ej., Anthropic, Cloudflare, Deepseek, OpenAI, Qwen), modelos locales de IA (a través de vLLM, Ray, etc.) y plataformas agregadoras de API (como Deepinfra, Openrouter y más).

**Características clave:**

- **Cambio dinámico de API de LLM y de agentes**: Alterna sin fricciones entre distintas APIs de LLM y agentes en tiempo real.
- **Compatibilidad integral de capacidades**: Soporte incorporado para Herramientas, Indicaciones, Recursos y métodos de Muestreo.
- **Agentes configurables**: Mayor flexibilidad con Herramientas seleccionables y personalizables desde la configuración del agente.
- **Control avanzado de muestreo**: Ajusta los parámetros de Muestreo y aprovecha el muestreo en múltiples rondas para resultados óptimos.
- **Compatibilidad multiplataforma**: Totalmente compatible con macOS, Windows y Linux.
- **Gratis y de código abierto (FOSS)**: Licencia permisiva que permite modificaciones y empaquetado personalizado de la aplicación.

**Más información:**

- [Documento de TUUI](https://www.tuui.com/)
- [Repositorio de GitHub de AIQL](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) es un asistente de codificación con agentes, de código abierto, para terminales.

**Funciones clave:**

- Compatibilidad total con Servidores MCP.
- Edita indicaciones con tu editor de texto preferido.
- Accede al instante a indicaciones guardadas con `@`.
- Controla y organiza Recursos de AWS directamente desde tu terminal.
- Herramientas, perfiles, gestión de contexto, compactación automática y mucho más.

**Primeros pasos**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) es un asistente de codificación con capacidades de agente y de código abierto para IDE.

**Características clave:**

- Compatibilidad con los IDE VSCode, JetBrains, Visual Studio y Eclipse.
- Controla y organiza los Recursos de AWS directamente desde tu IDE.
- Administra los permisos de cada Herramienta de MCP a través de la interfaz de usuario del IDE.

<div id="apify-mcp-tester">
  ### Apify MCP Tester
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) es un cliente de código abierto que se conecta a cualquier Servidor MCP mediante Server-Sent Events (SSE).
Es un Actor de Apify autónomo, diseñado para probar Servidores MCP sobre SSE, con soporte para encabezados de autorización.
Usa JavaScript sencillo (a la antigua) y está alojado en Apify, lo que te permite ejecutarlo sin configuración previa.

**Características clave:**

- Se conecta a cualquier Servidor MCP mediante SSE.
- Funciona con el [Apify MCP Server](https://apify.com/apify/actors-mcp-server) para interactuar con uno o más [Actors](https://apify.com/store) de Apify.
- Utiliza dinámicamente herramientas según el contexto y las consultas del usuario (si el servidor lo admite).

<div id="amp">
  ### Amp
</div>

[Amp](https://ampcode.com) es una herramienta de programación con capacidades agénticas creada por Sourcegraph. Se ejecuta en VS Code (y bifurcaciones compatibles como Cursor, Windsurf y VSCodium) y también como herramienta de línea de comandos. Además, es colaborativa en tiempo real: puedes compartir conversaciones y trabajar con tu equipo.

**Funciones clave:**

- Control granular sobre las Herramientas habilitadas y los permisos
- Compatibilidad con Servidores MCP definidos en `mcp.json` de VS Code

<div id="augment-code">
  ### Augment Code
</div>

[Augment Code](https://augmentcode.com) es una plataforma de desarrollo con IA para VS Code y JetBrains que incluye agentes autónomos, chat y autocompletado. Tanto los agentes locales como los remotos tienen conocimiento completo del código y compatibilidad nativa con el Protocolo de Contexto del Modelo (MCP), lo que permite ampliar el contexto mediante fuentes externas y herramientas.

**Funciones clave:**

- Compatibilidad completa con MCP en agentes locales y remotos.
- Añade contexto adicional a través de servidores MCP.
- Automatiza tus flujos de trabajo de desarrollo con herramientas MCP.
