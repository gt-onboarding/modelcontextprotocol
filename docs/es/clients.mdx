  ## Detalles del cliente

<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) es un asistente de escritorio de IA de código abierto y multiplataforma que admite Herramientas a través de Servidores MCP.

**Características clave:**

- Los Servidores MCP integrados se pueden habilitar y deshabilitar rápidamente.
- Los usuarios pueden agregar más servidores modificando el archivo de configuración.
- Es de código abierto y fácil de usar, adecuado para principiantes.
- El soporte futuro para MCP se mejorará continuamente.

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) es una biblioteca de Rust diseñada para simplificar la creación de agentes de IA. La biblioteca incluye integración fluida con Servidores MCP.

[Ejemplo de integración con Servidor MCP](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**Funciones clave:**

- Multi-LLM: admitimos la mayoría de las API de LLM (OpenAI, Anthropic, Gemini, Ollama y todas las compatibles con la API de OpenAI).
- Compatibilidad nativa con Servidores MCP.
- Crea flujos de agentes en un lenguaje con seguridad de tipos y de memoria como Rust.

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) es una plataforma de IA sin código que te ayuda a crear agentes que gestionan ventas, marketing y tareas creativas las 24 horas del día. Conecta de forma segura más de 2,500 API y más de 10,000 Herramientas mediante MCP.

**Funciones clave:**

- Creación de agentes de IA sin código y diseño de flujos de trabajo.
- Accede a una amplia biblioteca de más de 10,000 Herramientas y más de 2,500 API a través de MCP.
- Proceso sencillo de 3 pasos para conectar Servidores MCP.
- Gestiona conexiones de forma segura y revoca el acceso en cualquier momento.

**Más información:**

- [Integración de AgenticFlow con MCP](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] es una aplicación de chat de IA para escritorio, nativa y multiplataforma, con soporte para el Protocolo de Contexto de Modelo (MCP). Es compatible con múltiples proveedores de IA (p. ej., Anthropic, Cloudflare, Deepseek, OpenAI, Qwen), modelos de IA locales (mediante vLLM, Ray, etc.) y plataformas de API agregadas (como Deepinfra, Openrouter y más).

**Características clave:**

- **Cambio dinámico de API de LLM y agentes**: Cambia sin interrupciones entre diferentes API de LLM y agentes en tiempo real.
- **Compatibilidad integral de capacidades**: Compatibilidad integrada con Herramientas, Indicaciones, Recursos y métodos de Muestreo.
- **Agentes configurables**: Mayor flexibilidad con Herramientas seleccionables y personalizables desde la configuración del agente.
- **Control avanzado de muestreo**: Ajusta parámetros de Muestreo y aprovecha el muestreo en múltiples rondas para obtener resultados óptimos.
- **Compatibilidad multiplataforma**: Totalmente compatible con macOS, Windows y Linux.
- **Gratis y de código abierto (FOSS)**: La licencia permisiva permite modificaciones y empaquetado personalizado de la aplicación.

**Más información:**

- [Documentación de TUUI](https://www.tuui.com/)
- [Repositorio de GitHub de AIQL](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) es un asistente de codificación agentivo de código abierto para terminales.

**Características clave:**

- Compatibilidad total con Servidores MCP.
- Edita las indicaciones usando tu editor de texto preferido.
- Accede al instante a las indicaciones guardadas con `@`.
- Controla y organiza los recursos de AWS directamente desde tu terminal.
- Herramientas, perfiles, gestión de contexto, autocompactación y mucho más.

**Primeros pasos**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) es un asistente de programación de código abierto y con capacidades de agente para IDE.

**Funciones clave:**

- Compatibilidad con los IDE VSCode, JetBrains, Visual Studio y Eclipse.
- Controla y organiza los recursos de AWS directamente desde tu IDE.
- Gestiona los permisos de cada Herramienta MCP desde la interfaz de usuario del IDE.

<div id="apify-mcp-tester">
  ### Probador MCP de Apify
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) es un Cliente MCP de código abierto que se conecta a cualquier Servidor MCP mediante Eventos enviados por el servidor (SSE).
Es un Actor de Apify independiente, diseñado para probar Servidores MCP a través de SSE, con compatibilidad con encabezados de autorización.
Utiliza JavaScript simple (estilo clásico) y está alojado en Apify, lo que te permite ejecutarlo sin configuración previa.

**Funciones clave:**

- Se conecta a cualquier Servidor MCP mediante SSE.
- Funciona con el [Apify MCP Server](https://apify.com/apify/actors-mcp-server) para interactuar con uno o más [Actors](https://apify.com/store) de Apify.
- Utiliza Herramientas de forma dinámica según el contexto y las consultas del usuario (si el servidor lo admite).

<div id="amp">
  ### Amp
</div>

[Amp](https://ampcode.com) es una herramienta de programación con capacidades de agente creada por Sourcegraph. Funciona en VS Code (y bifurcaciones compatibles como Cursor, Windsurf y VSCodium) y como herramienta de línea de comandos. También es colaborativa: puedes compartir conversaciones y colaborar con tu equipo.

**Funciones clave:**

- Control granular sobre las Herramientas habilitadas y los permisos
- Compatibilidad con Servidores MCP definidos en el `mcp.json` de VS Code

<div id="augment-code">
  ### Augment Code
</div>

[Augment Code](https://augmentcode.com) es una plataforma de programación con IA para VS Code y JetBrains que ofrece agentes autónomos, chat y completados. Tanto los agentes locales como los remotos tienen conocimiento integral de la base de código y compatibilidad nativa con el Protocolo de Contexto de Modelo (MCP), lo que habilita un contexto enriquecido a través de fuentes externas y herramientas.
