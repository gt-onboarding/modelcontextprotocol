<div id="getting-started">
  ## Introducción
</div>

Cada SDK ofrece la misma funcionalidad, pero sigue los estándares y las mejores prácticas de su lenguaje. Todos los SDK admiten:

* Crear Servidores MCP que expongan Herramientas, Recursos e Indicaciones
* Crear Clientes MCP que puedan conectarse a cualquier Servidor MCP
* Protocolos de transporte locales y remotos
* Cumplimiento del protocolo con seguridad de tipos

Visita la página del SDK para el lenguaje que elijas para encontrar instrucciones de instalación, documentación y ejemplos.
