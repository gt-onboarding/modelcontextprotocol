<div id="getting-started">
  ## 入门
</div>

各个 SDK 提供的功能一致，但遵循各自语言的惯用法和最佳实践。所有 SDK 均支持：

* 创建公开工具（Tools）、资源（Resources）和提示模板（Prompts）的 MCP 服务器
* 构建可连接到任意 MCP 服务器的 MCP 客户端
* 本地与远程传输协议
* 具备类型安全的协议兼容性

访问所选语言的 SDK 页面以获取安装说明、文档和示例。
