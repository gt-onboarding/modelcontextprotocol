<div id="getting-started">
  ## 快速开始
</div>

<div id="using-reference-servers">
  ### 使用参考服务器
</div>

基于 TypeScript 的服务器可直接通过 `npx` 运行：

```bash
npx -y @modelcontextprotocol/server-memory
```

基于 Python 的服务器可通过 `uvx`（推荐）或 `pip` 使用：

```bash
# 使用 uvx
uvx mcp-server-git

# 使用 pip
pip install mcp-server-git
python -m mcp_server_git
```

<div id="configuring-with-claude">
  ### 在 Claude 中进行配置
</div>

要在 Claude 中使用 MCP 服务器，请将其添加到你的配置中：

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/path/to/allowed/files"
      ]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
