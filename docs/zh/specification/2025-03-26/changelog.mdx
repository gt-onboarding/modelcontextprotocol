title: 关键变更
---

本文档列出了自上一版 [2024-11-05](/zh/specification/2024-11-05) 以来对模型上下文协议（MCP）规范所做的变更。

<div id="major-changes">
  ## 重大变更
</div>

1. 基于 OAuth 2.1 的**[全面授权框架](/zh/specification/2025-03-26/basic/authorization)**（PR
   [#133](https://github.com/modelcontextprotocol/specification/pull/133)）
2. 以更灵活的**[可流式 HTTP 传输方式](/zh/specification/2025-03-26/basic/transports#streamable-http)**替代先前的 HTTP+SSE 传输（PR
   [#206](https://github.com/modelcontextprotocol/specification/pull/206)）
3. 新增对 JSON-RPC **[批处理](https://www.jsonrpc.org/specification#batch)** 的支持（PR
   [#228](https://github.com/modelcontextprotocol/specification/pull/228)）
4. 新增完善的**工具注解**，用于更好地描述工具行为，例如是否为只读或具有破坏性（PR
   [#185](https://github.com/modelcontextprotocol/specification/pull/185)）

<div id="other-schema-changes">
  ## 其他模式更改
</div>

* 在 `ProgressNotification` 中新增 `message` 字段，用于提供更具描述性的状态更新
* 新增对音频数据的支持，补充现有的文本和图像内容类型
* 新增 `completions` 功能，用于明确表示支持参数自动补全建议

参见
[更新后的模式](http://github.com/modelcontextprotocol/specification/tree/main/schema/2025-03-26/schema.ts)
了解更多详情。
