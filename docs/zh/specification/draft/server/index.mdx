服务器通过 MCP 为语言模型提供添加上下文的基础能力。这些基元使客户端、服务器与语言模型之间的交互更加丰富：

* **提示模板**：预定义的模板或指令，用于引导与语言模型的交互
* **资源**：为模型提供额外上下文的结构化数据或内容
* **工具**：允许模型执行操作或检索信息的可执行功能

每种基元可按以下控制层级概括：

| 基元 | 控制                | 描述                                      | 示例                         |
| ---- | ------------------- | ----------------------------------------- | ---------------------------- |
| 提示模板 | 用户控制         | 由用户选择触发的交互式模板                 | 斜杠命令、菜单选项            |
| 资源 | 应用控制            | 由客户端附加并管理的上下文数据             | 文件内容、git 历史            |
| 工具 | 模型控制            | 暴露给 LLM 以执行操作的功能                 | API POST 请求、文件写入       |
