Cette page présente divers serveurs Model Context Protocol (MCP) qui illustrent les capacités et la polyvalence du protocole. Ces serveurs permettent aux modèles de langage de grande taille (LLM) d’accéder en toute sécurité à des outils et à des sources de données.

<div id="reference-implementations">
  ## Implémentations de référence
</div>

Ces serveurs MCP de référence officiels illustrent les fonctionnalités essentielles de MCP et l’utilisation du SDK :

<div id="current-reference-servers">
  ### Serveurs de référence actuels
</div>

- **[Everything](https://github.com/modelcontextprotocol/servers/tree/main/src/everything)** - Serveur de référence/de test avec des invites, des ressources et des outils
- **[Fetch](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)** - Récupération et conversion de contenu web pour une utilisation efficace des LLM
- **[Filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)** - Opérations de fichiers sécurisées avec des contrôles d’accès configurables
- **[Git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)** - Outils pour lire, rechercher et manipuler des dépôts Git
- **[Memory](https://github.com/modelcontextprotocol/servers/tree/main/src/memory)** - Système de mémoire persistante basé sur un graphe de connaissances
- **[Sequential Thinking](https://github.com/modelcontextprotocol/servers/tree/main/src/sequentialthinking)** - Résolution de problèmes dynamique et introspective au moyen de séquences de réflexion
- **[Time](https://github.com/modelcontextprotocol/servers/tree/main/src/time)** - Fonctions de conversion de l’heure et des fuseaux horaires

<div id="getting-started">
  ## Bien démarrer
</div>

<div id="using-reference-servers">
  ### Utilisation des serveurs de référence
</div>

Les serveurs basés sur TypeScript peuvent être utilisés directement avec `npx` :

```bash
npx -y @modelcontextprotocol/server-memory
```

Les serveurs basés sur Python peuvent être utilisés avec `uvx` (recommandé) ou `pip` :

```bash
# Avec uvx
uvx mcp-server-git

# Avec pip
pip install mcp-server-git
python -m mcp_server_git
```

<div id="configuring-with-claude">
  ### Configuration avec Claude
</div>

Pour utiliser un Serveur MCP avec Claude, ajoutez-le à votre configuration :

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/path/to/allowed/files"
      ]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
