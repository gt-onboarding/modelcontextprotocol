---
title: Qu’est-ce que le Model Context Protocol (MCP) ?
sidebarTitle: Qu’est-ce que MCP ?
---

Le MCP (Model Context Protocol) est une norme open source qui permet de connecter les applications d’IA à des systèmes externes.

Avec le MCP, des applications d’IA comme Claude ou ChatGPT peuvent se connecter à des sources de données (p. ex. fichiers locaux, bases de données), à des outils (p. ex. moteurs de recherche, calculateurs) et à des workflows (p. ex. inviteurs spécialisés), leur permettant d’accéder à des informations clés et d’exécuter des tâches.

Pensez au MCP comme à un port USB‑C pour les applications d’IA. De la même manière que l’USB‑C offre un moyen standardisé de connecter des appareils électroniques, le MCP offre un moyen standardisé de connecter des applications d’IA à des systèmes externes.

<div id="what-can-mcp-enable">
  ## Que permet MCP ?
</div>

* Des agents peuvent accéder à votre Google Calendar et à Notion, faisant office d’assistant IA plus personnalisé.
* Claude Code peut générer une application web complète à partir d’une maquette Figma.
* Des chatbots d’entreprise peuvent se connecter à plusieurs bases de données au sein d’une organisation, permettant aux utilisateurs d’analyser des données via le chat.
* Des modèles d’IA peuvent créer des modèles 3D dans Blender et les imprimer avec une imprimante 3D.

<div id="why-does-mcp-matter">
  ## Pourquoi le MCP est-il important ?
</div>

Selon votre rôle dans l’écosystème, le MCP peut offrir de nombreux avantages.

* **Développeurs** : le MCP réduit le temps de développement et la complexité lors de la création d’une application ou d’un agent d’IA, ou de leur intégration.
* **Applications ou agents d’IA** : le MCP donne accès à un écosystème de sources de données, d’outils et d’applications qui renforcent les capacités et améliorent l’expérience utilisateur.
* **Utilisateurs finaux** : le MCP permet des applications ou agents d’IA plus performants, capables d’accéder à vos données et d’agir en votre nom lorsque nécessaire.
