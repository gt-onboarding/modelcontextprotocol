<div id="getting-started">
  ## Prise en main
</div>

Chaque SDK offre les mêmes fonctionnalités tout en respectant les idiomes et les meilleures pratiques de son langage. Tous les SDK prennent en charge :

* La création de serveurs MCP qui exposent des Outils, des Ressources et des Invites
* La création de clients MCP pouvant se connecter à n&#39;importe quel serveur MCP
* Des protocoles de transport locaux et distants
* La conformité au protocole avec sécurité des types

Consultez la page du SDK pour la langue de votre choix afin d’obtenir les instructions d’installation, la documentation et des exemples.
