  ## Détails du client
</div>

<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) est un assistant IA de bureau multiplateforme open source qui prend en charge les Outils via des Serveurs MCP.

**Fonctionnalités clés :**

- Des Serveurs MCP intégrés peuvent être rapidement activés et désactivés.
- Les utilisateurs peuvent ajouter d’autres serveurs en modifiant le fichier de configuration.
- Open source et convivial, il convient aux débutants.
- La prise en charge de MCP sera continuellement améliorée à l’avenir.

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) est une bibliothèque Rust conçue pour simplifier la création d’agents IA. La bibliothèque inclut une intégration transparente avec des serveurs MCP.

[Exemple d’intégration de serveur MCP](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**Fonctionnalités clés :**

- Multi-LLM – Nous prenons en charge la plupart des API de LLM (OpenAI, Anthropic, Gemini, Ollama et toutes les API compatibles OpenAI).
- Prise en charge intégrée des serveurs MCP.
- Créez des flux d’agents dans un langage sûr pour les types et la mémoire comme Rust.

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) est une plateforme d’IA sans code qui vous aide à créer des agents capables de gérer des tâches de vente, de marketing et de création 24h/24, 7j/7. Connectez plus de 2 500 API et plus de 10 000 Outils en toute sécurité via l’Hôte MCP.

**Fonctionnalités clés :**

- Création d’agents IA sans code et conception de workflows.
- Accédez à une vaste bibliothèque de plus de 10 000 Outils et plus de 2 500 API via l’Hôte MCP.
- Processus simple en 3 étapes pour connecter des Serveurs MCP.
- Gérez les connexions en toute sécurité et révoquez l’accès à tout moment.

**En savoir plus :**

- [Intégration MCP d’AgenticFlow](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] est une application de chat IA native, multiplateforme, avec prise en charge du Model Context Protocol (MCP). Elle prend en charge plusieurs fournisseurs d’IA (p. ex. Anthropic, Cloudflare, Deepseek, OpenAI, Qwen), des modèles IA locaux (via vLLM, Ray, etc.) ainsi que des plateformes d’API agrégées (telles que Deepinfra, OpenRouter, et plus encore).

**Fonctionnalités clés :**

- **Basculement dynamique d’API LLM et d’agents** : Passez instantanément d’une API LLM ou d’un agent à un autre.
- **Prise en charge complète des capacités** : Prise en charge intégrée des Outils, de l’Invite, des Ressources et des méthodes d’Échantillonnage.
- **Agents configurables** : Flexibilité accrue grâce à des outils sélectionnables et personnalisables via les paramètres d’agent.
- **Contrôle avancé de l’échantillonnage** : Modifiez les paramètres d’échantillonnage et exploitez l’échantillonnage multi-itérations pour des résultats optimaux.
- **Compatibilité multiplateforme** : Entièrement compatible avec macOS, Windows et Linux.
- **Libre et open source (FOSS)** : Une licence permissive autorise les modifications et le reconditionnement d’applications personnalisées.

**En savoir plus :**

- [Documentation TUUI](https://www.tuui.com/)
- [Dépôt GitHub AIQL](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) est un assistant de codage agentique open source pour terminaux.

**Fonctionnalités clés :**

- Prise en charge complète des Serveurs MCP.
- Modifiez les invites à l’aide de votre éditeur de texte préféré.
- Accédez instantanément aux invites enregistrées avec `@`.
- Contrôlez et organisez les Ressources AWS directement depuis votre terminal.
- Outils, profils, gestion du contexte, compactage automatique, et bien plus encore !

**Bien démarrer**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) est un assistant de développement agentique open source pour les IDE.

**Fonctionnalités clés :**

- Prise en charge des IDE VSCode, JetBrains, Visual Studio et Eclipse.
- Contrôlez et organisez les Ressources AWS directement depuis votre IDE.
- Gérez les autorisations pour chaque Outil MCP via l’interface utilisateur de l’IDE.

<div id="apify-mcp-tester">
  ### Apify MCP Tester
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) est un client open source qui se connecte à n’importe quel Serveur MCP via des événements envoyés par le serveur (SSE).
C’est un Actor Apify autonome conçu pour tester des Serveurs MCP via SSE, avec prise en charge des en-têtes d’autorisation.
Il utilise du JavaScript simple (à l’ancienne) et est hébergé sur Apify, ce qui vous permet de l’exécuter sans aucune configuration.

**Fonctionnalités clés :**

- Se connecte à n’importe quel Serveur MCP via SSE.
- Fonctionne avec l’[Apify MCP Server](https://apify.com/apify/actors-mcp-server) pour interagir avec un ou plusieurs [Actors](https://apify.com/store) Apify.
- Utilise dynamiquement des Outils en fonction du contexte et des requêtes utilisateur (si pris en charge par le serveur).

<div id="amp">
  ### Amp
</div>

[Amp](https://ampcode.com) est un outil de codage agentique développé par Sourcegraph. Il s’exécute dans VS Code (et des forks compatibles comme Cursor, Windsurf et VSCodium) ainsi qu’en ligne de commande. Il est également multi-utilisateur — vous pouvez partager des fils de discussion et collaborer avec votre équipe.

**Fonctionnalités clés :**

- Contrôle granulaire des outils activés et des autorisations
- Prise en charge des serveurs MCP définis dans le `mcp.json` de VS Code

<div id="augment-code">

<div id="zencoder">
  ### Zencoder
</div>

[Zencoder](https://zecoder.ai) est un agent de développement disponible sous forme d’extension pour VS Code et la famille d’IDE JetBrains, qui s’intègre là où les développeurs travaillent déjà. Il inclut RepoGrokking (compréhension contextuelle approfondie du code), un pipeline agentique et la possibilité de créer et de partager des agents personnalisés.

**Fonctionnalités clés :**

- RepoGrokking - compréhension contextuelle approfondie du code
- Pipeline agentique - exécute, teste et valide le code avant de produire un résultat
- Plateforme Zen Agents - possibilité de concevoir et de créer des agents personnalisés et de les partager avec l’équipe
- Bibliothèque d’Outils MCP intégrée, avec installations en un clic
- Agents spécialisés pour les tests unitaires et de bout en bout (E2E)

**En savoir plus :**

- [Documentation Zencoder](https://docs.zencoder.ai)

Si vous avez ajouté la prise en charge de MCP à votre application, nous vous encourageons à soumettre une pull request pour l’ajouter à cette liste. Avantages de l’ajout de la prise en charge de MCP :

- Offrir aux utilisateurs des options d’intégration flexibles
