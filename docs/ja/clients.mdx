<div id="5ire">
  ### 5ire
</div>

[5ire](https://github.com/nanbingxyz/5ire) は、MCPサーバー経由でツールを利用できる、オープンソースのクロスプラットフォーム対応デスクトップAIアシスタントです。

**主な機能:**

- 内蔵のMCPサーバーをすばやく有効化／無効化できます。
- 設定ファイルを編集してサーバーを追加できます。
- オープンソースで使いやすく、初心者にも適しています。
- MCP対応は今後も継続的に拡充・改善されます。

<div id="agentai">
  ### AgentAI
</div>

[AgentAI](https://github.com/AdamStrojek/rust-agentai) は、AIエージェントの作成を容易にするために設計されたRustライブラリです。MCPサーバーとのシームレスな統合を備えています。

[MCPサーバー統合の例](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**主な機能:**

- マルチLLM — 主要なLLM API（OpenAI、Anthropic、Gemini、Ollama、およびOpenAI API互換の各種）をサポート。
- MCPサーバーのサポートを内蔵。
- Rustのような型安全かつメモリ安全な言語でエージェントのフローを作成可能。

<div id="agenticflow">
  ### AgenticFlow
</div>

[AgenticFlow](https://agenticflow.ai/) は、営業、マーケティング、クリエイティブの業務を24時間自動でこなすエージェントをノーコードで構築できるAIプラットフォームです。MCP経由で2,500以上のAPIと10,000以上のツールに安全に接続できます。

**主な特長:**

- ノーコードでのAIエージェント作成とワークフロー構築。
- MCPを通じて、10,000以上のツールと2,500以上のAPIからなる豊富なライブラリにアクセス。
- MCPサーバー接続はシンプルな3ステップ。
- 接続を安全に管理し、アクセスはいつでも取り消し可能。

**詳細はこちら:**

- [AgenticFlow MCP Integration](https://agenticflow.ai/mcp)

<div id="aiql-tuui">
  ### AIQL TUUI
</div>

[AIQL TUUI] は、MCP対応のネイティブかつクロスプラットフォームなデスクトップAIチャットアプリです。複数のAIプロバイダー（例：Anthropic、Cloudflare、Deepseek、OpenAI、Qwen）、ローカルAIモデル（vLLM、Ray など経由）、および集約型APIプラットフォーム（Deepinfra、Openrouter など）をサポートします。

**主な機能:**

- **動的なLLM APIとエージェントの切り替え**: さまざまなLLM APIやエージェントをシームレスにオンザフライで切り替え可能。
- **包括的な機能サポート**: ツール、プロンプト、リソース、サンプリング手法を標準でサポート。
- **設定可能なエージェント**: エージェント設定からツールを選択・カスタマイズでき、高い柔軟性を実現。
- **高度なサンプリング制御**: サンプリングパラメータを調整し、最適化のためにマルチラウンドサンプリングを活用可能。
- **クロスプラットフォーム対応**: macOS、Windows、Linuxで完全互換。
- **無料・オープンソース（FOSS）**: 寛容なライセンスにより改変やカスタムアプリのバンドルが可能。

**詳細はこちら:**

- [TUUI document](https://www.tuui.com/)
- [AIQL GitHub repository](https://github.com/AI-QL)

<div id="amazon-q-cli">
  ### Amazon Q CLI
</div>

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) は、ターミナル向けのオープンソースのエージェント型コーディングアシスタントです。

**主な機能:**

- MCPサーバーをフルサポート。
- お好みのテキストエディタでプロンプトを編集。
- `@` で保存済みプロンプトに即アクセス。
- ターミナルから直接、AWSのリソースを管理・整理。
- ツール、プロファイル、コンテキスト管理、自動コンパクト化など、さらに豊富な機能！

**はじめに**

```bash
brew install amazon-q
```

<div id="amazon-q-ide">
  ### Amazon Q IDE
</div>

[Amazon Q IDE](https://aws.amazon.com/q/developer) は、IDE 向けのオープンソースのエージェント型コーディングアシスタントです。

**主な機能:**

- VS Code、JetBrains、Visual Studio、Eclipse の各 IDE をサポート。
- IDE から直接 AWS リソースを操作・管理。
- IDE のユーザーインターフェースから各 MCP ツールの権限を管理。

<div id="apify-mcp-tester">
  ### Apify MCP Tester
</div>

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) は、サーバー送信イベント（SSE）を用いて任意のMCPサーバーに接続できるオープンソースのクライアントです。
SSE経由でMCPサーバーをテストするために設計されたスタンドアロンのApify Actorで、Authorizationヘッダーにも対応しています。
プレーンなJavaScript（昔ながらのスタイル）で実装され、Apify上でホストされているため、セットアップ不要で実行できます。

**主な機能:**

- SSE経由で任意のMCPサーバーに接続。

<div id="adding-mcp-support-to-your-application">
  ## アプリケーションへのMCP対応の追加
</div>

アプリケーションにMCP対応を追加した場合は、このリストへの掲載に向けて、ぜひPull Requestを送ってください。MCPの統合により、ユーザーに強力なコンテキスト対応のAI機能を提供でき、アプリケーションを拡大中のMCPエコシステムの一員にできます。

MCP対応を追加する利点:

- ユーザーが自身のコンテキストとツールを持ち込めるようにする
- 相互運用可能なAIアプリケーションが拡大するエコシステムに参加する
- ユーザーに柔軟な統合オプションを提供する
- ローカルファーストのAIワークフローをサポートする

アプリケーションでMCPの実装を始めるには、[Python](https://github.com/modelcontextprotocol/python-sdk) または [TypeScript SDKのドキュメント](https://github.com/modelcontextprotocol/typescript-sdk) をご覧ください
