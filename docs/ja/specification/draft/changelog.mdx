このドキュメントでは、前回の改訂版である[2025-06-18](/ja/specification/2025-06-18)以降にModel Context Protocol（MCP）仕様に加えられた変更を一覧します。

<div id="major-changes">
  ## 主要な変更点
</div>

1. 認可サーバーの探索を強化し、[OpenID Connect Discovery 1.0](https://openid.net/specs/openid-connect-discovery-1_0.html) に対応しました（PR [#797](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/797)）。
2. サーバーがツール、リソース、プロンプトに関する追加メタデータとしてアイコンを公開できるようにしました（[SEP-973](https://github.com/modelcontextprotocol/modelcontextprotocol/issues/973)）。
