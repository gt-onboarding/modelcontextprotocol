サーバーは、MCPを通じて言語モデルにコンテキストを付与するための基本的な構成要素を提供します。これらのプリミティブにより、クライアント、サーバー、言語モデル間で豊かなやり取りが可能になります。

* **プロンプト**: 言語モデルとのやり取りを導くための事前定義されたテンプレートや指示
* **リソース**: モデルに追加のコンテキストを与える構造化データやコンテンツ
* **ツール**: モデルがアクションを実行したり情報を取得したりするための実行可能な関数

各プリミティブは、次の制御階層で要約できます。

| プリミティブ | 制御                  | 説明                                              | 例                              |
| ------------ | --------------------- | ------------------------------------------------- | ------------------------------- |
| プロンプト   | ユーザー制御          | ユーザーの選択で呼び出されるインタラクティブなテンプレート | スラッシュコマンド、メニューオプション |
| リソース     | アプリケーション制御  | クライアントによって添付・管理されるコンテキストデータ | ファイル内容、Git履歴           |
| ツール       | モデル制御            | LLMに公開され、アクションの実行に用いられる関数   | APIのPOSTリクエスト、ファイル書き込み |

以下で、これらの主要なプリミティブをさらに詳しく確認してください。
