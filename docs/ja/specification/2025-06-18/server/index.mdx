<Info>**プロトコル改訂**: 2025-06-18</Info>

サーバーは、Model Context Protocol（MCP）を通じて言語モデルにコンテキストを付与するための基本的な構成要素を提供します。これらのプリミティブにより、クライアント、サーバー、言語モデル間で豊かなやり取りが可能になります。

* **プロンプト**: 言語モデルとの対話を導くための事前定義のテンプレートや指示
* **リソース**: モデルに追加のコンテキストを与える構造化データやコンテンツ
* **ツール**: モデルがアクションの実行や情報の取得に用いる実行可能な関数

各プリミティブは次の制御階層で要約できます。

| プリミティブ | 制御                     | 説明                                               | 例                               |
| ------------ | ------------------------ | -------------------------------------------------- | -------------------------------- |
| プロンプト   | ユーザー制御             | ユーザーの選択で呼び出される対話型テンプレート     | スラッシュコマンド、メニュー項目 |
| リソース     | アプリケーション制御     | クライアントが添付・管理するコンテキストデータ     | ファイル内容、git履歴            |
| ツール       | モデル制御               | アクション実行のためにLLMに公開される関数          | APIのPOSTリクエスト、ファイル書き込み |
