<Info>**プロトコル改訂**: 2025-03-26</Info>

サーバーは、MCP を通じて言語モデルにコンテキストを付与するための基本的な構成要素を提供します。これらのプリミティブにより、クライアント、サーバー、言語モデル間で豊かな相互作用が可能になります。

* **プロンプト**: 言語モデルとのやり取りを導くための事前定義済みテンプレートまたは指示
* **リソース**: モデルに追加のコンテキストを提供する構造化データやコンテンツ
* **ツール**: モデルがアクションを実行したり情報を取得したりできる実行可能な関数

各プリミティブは、次のコントロール階層で要約できます。

| プリミティブ | コントロール           | 説明                                             | 例                               |
| ------------ | ---------------------- | ------------------------------------------------ | -------------------------------- |
| プロンプト   | ユーザー制御           | ユーザーの選択で呼び出されるインタラクティブなテンプレート | スラッシュコマンド、メニューオプション |
| リソース     | アプリケーション制御   | クライアントによって添付・管理されるコンテキストデータ | ファイル内容、git 履歴            |
| ツール       | モデル制御             | LLM に公開され、アクションの実行に用いられる関数   | API の POST リクエスト、ファイル書き込み |
