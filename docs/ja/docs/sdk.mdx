<div id="getting-started">
  ## はじめに
</div>

各SDKは同等の機能を提供しますが、各言語のイディオムやベストプラクティスに沿った設計になっています。すべてのSDKは次をサポートします:

* ツール、リソース、プロンプトを公開するMCPサーバーの作成
* 任意のMCPサーバーに接続できるMCPクライアントの構築
* ローカルおよびリモートのトランスポートプロトコル
* 型安全性を備えたプロトコル準拠

利用したい言語のSDKページで、インストール手順、ドキュメント、サンプルをご確認ください。
