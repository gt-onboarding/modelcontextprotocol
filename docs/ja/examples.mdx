このページでは、Model Context Protocol (MCP) の多様なサーバーを紹介し、プロトコルの機能と柔軟性を示します。これらのサーバーにより、Large Language Models (LLMs) はツールやデータソースに安全にアクセスできます。

<div id="reference-implementations">
  ## リファレンス実装
</div>

これらの公式リファレンスサーバーは、MCPのコア機能とSDKの使い方を示すものです。

  ### 現在のリファレンスサーバー

- **[Time](https://github.com/modelcontextprotocol/servers/tree/main/src/time)** - 時刻およびタイムゾーンの変換機能

  ## はじめに
</div>

<div id="using-reference-servers">
  ### リファレンスサーバーの利用
</div>

TypeScript ベースのサーバーは `npx` で直接実行できます:

```bash
npx -y @modelcontextprotocol/server-memory
```

Python ベースのサーバーは `uvx`（推奨）または `pip` で利用できます:

```bash
# uvx を使用
uvx mcp-server-git

# pip を使用
pip install mcp-server-git
python -m mcp_server_git
```

<div id="configuring-with-claude">
  ### Claudeでの設定
</div>

ClaudeでMCPサーバーを利用するには、設定に次の内容を追加します：

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/path/to/allowed/files"
      ]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
